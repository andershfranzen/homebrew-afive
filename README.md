# Homebrew tap for afive apps

Install Andromeda:

```sh
brew tap andershfranzen/afive
brew install --cask andromeda
```

Andromeda is a macOS menu-bar controller for XREAL One Pro glasses.

Install Pier:

```sh
brew tap andershfranzen/afive
brew install --cask pier
```

Pier is a macOS menu-bar Docker fleet monitor over SSH.
